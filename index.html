About Me

I am Sneha Pal, a 3rd-year Computer Science and Engineering student at Maharshi Dayanand University, Delhi, with a focus on front-end development and user experience design. I am passionate about the ever evolving world of technology and its potential to shape the future. I am a front-end developer with a keen eye for design and a strong foundation in coding, striving to create intuitive and visually appealing user experiences.

SKILLS

HTML5

I created a resume project using only HTML, focusing on simplicity and structure. This project demonstrates how essential content can be effectively presented without relying on additional styling languages like CSS or JavaScript. It includes all necessary sections such as personal information, education, skills, and work experience, providing a clear and organized layout. Through this project, I honed my understanding of HTML elements and their roles in structuring web content.

CSS3

I developed a project for a web design agency, utilizing CSS to create a visually appealing and professional layout. This project highlights the importance of aesthetics in web development, focusing on responsive design, typography, and color schemes. By applying CSS properties, I was able to enhance the user interface and overall user experience, ensuring the website is both functional and visually engaging. This project helped me refine my skills in CSS and responsive web design techniques.

BOOTSTRAP

I created a project called "TinDog" , a playful and modern website for dog lovers, using Bootstrap. This project showcases the power of Bootstrap’s pre-built components and grid system, allowing for a fully responsive and sleek design. With minimal custom CSS, I was able to incorporate features like navigation bars, cards, and carousels, ensuring an engaging user experience across all devices. Through this project, I strengthened my knowledge of Bootstrap’s framework, focusing on creating clean, mobile-first web designs efficiently.
